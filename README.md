# Netflix AI & Data Analysis Project 🎬🤖

## Overview
This project combines Data Analysis and Machine Learning techniques to explore and analyze the Netflix dataset.

The project includes:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- AI / Machine Learning Model

The goal is to discover insights and build predictive analysis using Python.

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

---

## Dataset Features
The dataset contains:
- Movies & TV Shows
- Directors & Cast
- Countries
- Ratings
- Genres
- Release Years
- Duration
- Descriptions

---

## Project Tasks

### Task 1: Data Understanding
- Dataset exploration
- Understanding columns and data types
- Statistical overview

### Task 2: Data Cleaning & Preprocessing
- Handling missing values
- Removing duplicates
- Formatting columns

### Task 3: Exploratory Data Analysis
- Movies vs TV Shows analysis
- Genre analysis
- Rating analysis
- Release year trends

### Task 4: Data Visualization
- Charts and graphs
- Content distribution
- Trend analysis

### Task 5: AI / Machine Learning Model
- Data preparation
- Feature selection
- Model training
- Prediction and evaluation
